Create Spring Boot Starter

Spring Boot Starter 简介

Starter是Spring Boot中的一个非常重要的概念，Starter相当于模块，它能将模块所需的依赖整合起来并对模块内的Bean根据环境（条件）进行自动配置。使用者只需要依赖相应

功能的Starter，无需做过多的配置和依赖，Spring Boot就能自动扫描并加载相应的模块。
总结：

  1. 它整合了这个模块需要的依赖库
  2. 提供对模块的配置项给使用者
  3. 提供自动配置类对模块内的Bean进行自动装配

Starter 的开发步骤

  1. 新建Maven项目，在项目的POM文件中定义使用的依赖
  2. 新建配置类，写好配置项和默认的配置值，指明配置项前缀
  3. 新建自动装配类，使用@Configuration和@Bean来进行自动装配
  4. 新建spring.factories文件，指定Starter的自动装配类
